# Interview Prep Audit Prompt

Use in a **separate conversation**. Paste this prompt and attach the interview prep files to review.
Update the project list as new projects are completed.

Files live in `notes/interview-prep/en/` (English) and `notes/interview-prep/es/` (Spanish).
Current files: `angular.md`, `typescript.md`, `architecture.md`, `general.md`, `javascript.md`, `css.md`, `git.md`, `sql.md`. Both folders must always stay in sync.

---

```
You are reviewing my interview preparation files for a junior Angular + Java Spring Boot
position at Spanish IT consultancies (NTT Data, Capgemini, Indra, and similar) in 2026.

I am Victor, 31 years old, no professional dev experience, first tech job search.
My stack: Angular + Java Spring Boot.

My projects:
- 01: todo list — components, signals, services, directives
- 02: weather app — HttpClient, RxJS, forkJoin, API integration
- 03: expense tracker — reactive forms, routing, localStorage, smart/dumb pattern
- 04: meal finder — route params, ActivatedRoute, effect(), favourites
- 05: task manager — Angular Material, MatTable, MatDialog, coordinator pattern
- 06: HR portal — route guards, lazy loading, HTTP interceptors, role-based access, CanDeactivate

Analyse everything above and give me a report with four sections:

**1. Missing topics**
Topics I should have files for but don't, given my stack and target companies.
For each missing topic, explain in one sentence why a Spanish consultancy would ask about it.

**2. Weak answers**
Answers that are too vague, too theoretical, or that don't reference my real projects.
Quote the weak part and explain what is missing.

**3. Imbalances**
Are there too many conceptual questions and not enough decision-based or pressure questions?
Give me the count per type and tell me what to fix.
Target ratio: 55% conceptual / 35% decision-based / 10% pressure.

**4. Suggested new questions**
3 to 5 new questions I don't have yet, based on gaps you found.
Write them in the exact format of my existing files:

**Question?**
What they really want to know: one sentence.
A: interview-ready answer with project example.
Red flag answer: what a weak candidate would say and why it fails.
```
