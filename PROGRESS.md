## Angular — Concepts to learn

### Basics
- [ ] Components: `@Component`, selector, template, styles
- [ ] Data binding: interpolation `{{ }}`, property binding `[]`, event binding `()`
- [ ] Two-way binding: `[(ngModel)]`
- [ ] Structural directives: `@if`, `@for`, `@switch` (new Angular 17+ syntax)
- [ ] Inputs and Outputs: `@Input()`, `@Output()`, `EventEmitter`
- [ ] Services and dependency injection: `@Injectable`, `inject()`
- [ ] Basic routing: `RouterModule`, `routerLink`, `RouterOutlet`

### Intermediate
- [ ] Signals: `signal()`, `computed()`, `effect()`
- [ ] HTTP Client: `HttpClient` + basic RxJS (`subscribe`, `pipe`, `map`, `catchError`)
- [ ] Reactive forms: `FormGroup`, `FormControl`, `Validators`
- [ ] Lifecycle hooks: `ngOnInit`, `ngOnDestroy`, `ngOnChanges`
- [ ] Pipes: built-in (`date`, `currency`, `async`) and custom pipes
- [ ] Route guards: `CanActivate`
- [ ] Lazy loading routes

### Advanced
- [ ] Standalone components (architecture without NgModules)
- [ ] State management with signals or NgRx
- [ ] HTTP Interceptors
- [ ] Basic testing: Jasmine + TestBed
- [ ] OnPush change detection
