## Angular — Concepts to learn *(concept = an idea or topic)*

### Basics *(basics = the most important and simple things)*
- [ ] Components: `@Component`, selector, template, styles
- [ ] Data binding: interpolation `{{ }}`, property binding `[]`, event binding `()`
- [ ] Two-way binding: `[(ngModel)]`
- [ ] Structural directives: `@if`, `@for`, `@switch` (new Angular 17+ syntax)
- [ ] Inputs and Outputs: `@Input()`, `@Output()`, `EventEmitter`
- [ ] Services and dependency injection: `@Injectable`, `inject()`
- [ ] Basic routing: `RouterModule`, `routerLink`, `RouterOutlet`

### Intermediate *(intermediate = between basic and advanced)*
- [ ] Signals: `signal()`, `computed()`, `effect()`
- [ ] HTTP Client: `HttpClient` + basic RxJS (`subscribe`, `pipe`, `map`, `catchError`)
- [ ] Reactive forms: `FormGroup`, `FormControl`, `Validators`
- [ ] Lifecycle hooks: `ngOnInit`, `ngOnDestroy`, `ngOnChanges`
- [ ] Pipes: built-in (`date`, `currency`, `async`) and custom pipes
- [ ] Route guards: `CanActivate`
- [ ] Lazy loading routes

### Advanced *(advanced = difficult, high level)*
- [ ] Standalone components (architecture without NgModules)
- [ ] State management with signals or NgRx
- [ ] HTTP Interceptors
- [ ] Basic testing: Jasmine + TestBed
- [ ] OnPush change detection

## SQL — Concepts to learn *(SQL = the language used to work with databases)*

### Basics
- [ ] SELECT, FROM, WHERE — reading data from a table *(table = a set of data organised in rows and columns)*
- [ ] ORDER BY, LIMIT — sorting and limiting results
- [ ] INSERT, UPDATE, DELETE — creating, modifying and removing data
- [ ] JOINs: INNER JOIN, LEFT JOIN — combining data from two tables
- [ ] GROUP BY, COUNT, SUM, AVG — grouping and calculating data
- [ ] Primary key, foreign key *(key = a field that identifies or links records)*

### Intermediate
- [ ] Subqueries *(subquery = a query inside another query)*
- [ ] Indexes — how to make queries faster
- [ ] Transactions: COMMIT, ROLLBACK

> **Note:** In Spain, most junior jobs use MySQL or PostgreSQL. Practice with one of them.
